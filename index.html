How does Rotate Group Mode work?

It works by connecting devices through the Multipeer Connectivity framework which connects devices by bluetooth, WiFi, or cellular connection. The app will choose whichever method works the fastest. Once devices are connected, the Host must order each device so that each picture goes in the correct place once everyone captures each picture. Then the Host taps the circle at the bottom of the screen to send a signal to each connected device which makes everyone's device take a picture.

How is this different from Boomerang?

This is different from Boomerang because the Rotate app uses multiple phones to capture each frame. Even while using just one phone to capture a rotate, each frame can be individualy picked to allow for a seamless burst video comprised of each picture.
